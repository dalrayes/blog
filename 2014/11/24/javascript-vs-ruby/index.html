Javascript vs Ruby

Looping Through!

A week into using Javascript and I miss Ruby already. Ruby is more intuitive and to the point, but Javascript feels more verbose, like something from English literature

Below, is an example of an annoying loop using ruby.


  loopy = true

  while loopy == true
    puts "Are we there yet?"
    answer = gets.chomp

    if answer == 'yes'
      puts "Finally!"

    elsif answer == 'no'
      puts "nooo"

    else answer == 'stop'
      loopy == false
      break

    end
  end

On the other hand, if we were to do something similar in Javascript it would look something like this:


  loopy = true

  while(loopy)
  {
    var answer = prompt("Are we there yet?")
      if(answer == "yes") {
        alert("Finally!!!");
      }
      else if (answer == "no") {
        alert("Nooo!!!");
      }
      else {
      loopy = false;
      }
  }

Although they look fairly similar, Javascript requires much more syntax (brackets and colons) than ruby. For looping, in addition to while, for and for...in loops, ruby provides methods until, each, and times.
